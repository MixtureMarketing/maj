# Modern Majestic PDF Catalog Map 📄

Follow this guide to organize your downloadable PDF catalogs. Place the files in the `/public/pdf/` folder with the specified names so they link correctly to the "Download PDF" buttons on the website.

---

## 📂 Catalog Files (`/public/pdf/`)

- `kitchen-cabinets.pdf`: Main catalog for European Kitchen Cabinets.
- `acrylic-panels.pdf`: Catalog for Decorative Acrylic Panels & Surfaces.
- `interior-doors.pdf`: Porta 2022 / Interior Doors Collection.
- `impact-doors.pdf`: Hurricane Impact / Exterior Doors catalog.
- `custom-closets.pdf`: Bespoke Closets & Storage solutions.

---

### 💡 How to Update:
1.  **Drop Files**: Copy your PDF files into `public/pdf/`.
2.  **Rename**: Ensure they match the filenames above exactly (lower-case, hyphens).
3.  **Automatic Linking**: The website is pre-configured to look for these paths (e.g., `/pdf/kitchen-cabinets.pdf`).

---

### 🚀 Pro Tip:
If you have multiple documents for a single category (e.g., technical sheets + design guides), you can create subfolders like `/public/pdf/technical/` and update the links in the page code accordingly.
